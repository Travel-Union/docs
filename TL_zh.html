II. 参与计划

  1. 本计划面向持有myTU个人账户的客户。您在使用账户的过程中即可参与–无需单独注册或申请。
  2. #TIME 以单一余额的形式与您的myTU账户关联，并在应用程序中与您的资金分开显示。
  3. #TIME 仅属于您本人。它不能被出售、让与、转让或发送给任何其他人或账户，也不能与他人的 #TIME 合并。

III. 赚取 #TIME

  1. 当您使用您的myTU个人支付卡为购买付款时，您会自动赚取 #TIME。
  2. 现金提取、账户之间的转账或非卡片购买的操作不会赚取 #TIME。商务卡不会赚取 #TIME。
  3. 符合条件的购买所授予的 #TIME 数额由我们设定，并可能随时更改。您随时可以在应用程序中查看您已赚取的 #TIME。
  4. 我们也可能不时为其他活动或促销授予 #TIME，具体按我们针对每项活动公布的条款执行。
  5. #TIME 在相关卡片交易处理完成后入账。对于此后被取消、退款、撤销，或被发现存在错误或欺诈的交易，我们可能会扣留或撤销与之相关的 #TIME。

IV. #TIME 的性质

  1. #TIME 没有现金价值。它不能被兑换、赎回或提取为现金，也不产生利息。
  2. 您对 #TIME 不享有任何所有权权益，也没有任何金钱性的请求权。#TIME 仅代表在本计划可用期间获得本条款所述福利的权利。
  3. #TIME 只能按照下文第五节的规定使用，并且只能在myTU内部使用。它不能在myTU应用程序以外的任何第三方处使用或通过其赎回。

V. 使用 #TIME

  1. 您可以在 myTU Outlet（应用程序内的商店）中使用 #TIME，以获取我们在那里提供的物品，例如代金券、代码和其他数字物品。
  2. 在您确认购买之前，Outlet 中的每件物品都会显示其价格。一件物品的定价可以是 #TIME、欧元，或 #TIME 与欧元的组合。要获取该物品，您需按所列金额支付显示的价格。
  3. 当您完成购买时，所显示的 #TIME（以及任何欧元）会从您的余额中扣除，物品–例如代金券或代码–会在应用程序中交付给您。
  4. Outlet 物品受购买时为每件物品所显示的具体条款约束，包括发行代金券或提供相关商品或服务的第三方的任何条款。
  5. 可获取的物品，以及它们所需的 #TIME 和欧元数额，会随时间变化。我们可能会随时添加、更改或移除物品。

VI. 计划的更改、暂停和终止

  1. 我们可能会随时按照第6条更改、暂停或终止本计划，或更改 #TIME 的赚取或使用方式。
  2. 如果我们终止本计划，我们将提前告知您，并告知您所持有的任何 #TIME 仍可使用的截止日期。到该日期仍未使用的 #TIME 将被取消，并将不再具有价值。
  3. 在您的myTU账户开立且状态良好期间，#TIME 持续可用。如果您的账户因任何原因被关闭或终止，任何剩余的 #TIME 将被取消，不具有价值，且不予退还或补偿。
  4. 如果您违反本条款或通用服务条款、滥用本计划、提供虚假或误导性信息，或参与欺诈或其他非法活动，我们可能会取消您的 #TIME 和/或暂停您对本计划的参与。
